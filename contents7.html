Web 자동화 처리 솔루션


개요

스크래퍼 및 크롤링 기술을 적용한 스크래퍼 솔루션을 이용하여, 인터넷상의 특정 키워드 또는 위치, 메뉴 등의
컨텐츠 자료를 주기적으로 수집 하거나, SCM사이트의 주문처리 등의 업무와 같이 반복적이고, 주기적으로 해야하는 업무처리를
스크래퍼 솔루션을 이용하여 자동으로 해당 업무등을 처리 하여 업무의 인력의 감축 및 업무의 효율성과 편의를 제공하는 서비스

Web 자동화 처리 기술

·   Web 사이트 자동 로그인

- 설정을 통해 Web사이트 자동 로그인 및 2차 인증 가능

·   스케쥴

- 특정 시간, 주기를 설정하여 원하는 시기에 따라 설정해놓은 업무 및 자료 수집을 자동으로 처리

·   Web 자동화 처리

- SCM의 주문 처리와 같이 반복적이고, 주기적으로 해야하는 업무를 Web 자동화 처리 솔루션을 통해 처리하고,
사용자가 원하는 결과셋을 수집하여 주문정보의 검토 및 이후 처리까지 자동화 지원

·   다양한 플랫폼 적용

- Web상 여러 사이트를 설정과, 간단한 셋팅을 통해 원하는 정보를 수집하거나 처리 가능

·   처리 이력 및 상태 체크

- 자동화 솔루션이 설정된 업무를 처리하는 도중, 오류 및 예외 상황 발생시 해당 오류내용 및 재처리 기능을 통해
설정된 업무를 수행 할 수 있도록 제공
